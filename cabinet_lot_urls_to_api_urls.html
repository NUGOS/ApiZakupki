Cabinet lot urls to api urls converter

GET remote/api/v2/by_lot/{id}

GET remote/api/v2/by_lot/{id}

id - purchase lot identifier (int)

allows to get full api url by lot id

request headers:

  • Api-Agent - type: string, reuqired
  • Acc-Token - type: string, required

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

possible data fields:

  • url - type: str

Examples:

request:

curl --header "Acc-Token: eyJlbXBsb3llZSI6NjEyNjEsIm1lcmNoYW50Ijo1NTEzOH0.C8di6Q.JPtyyp32oH7KaM7WqyjQPE0SoXA" -X GET "http://my.example.com:5001/remote/api/v2/by_lot/12312"

response:

{
    "status": "ok",
    "errors": [],
    "data": {
        "url": "remote/api/v2/entity/274073/lot/52494"
    },
    "system": ""
}
