Suggest заказчиков

GET remote/api/v1/merchant_suggest?q={search_phrase}

GET remote/api/v2/merchant_suggest?q={search_phrase}

v2 - add support to suggest commercial merchants

request headers:

  • Api-Agent - type: string, reuqired

возвращает список возможных заказчиков, удовлетворяющих search_phrase

request headers:

  • Acc-Token - type: string, required

possible request arguments:

  • q - type: string, required

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

possible response data fields:

  • merchants - type: array of merchant_suggest objects or empty, described below

possible merchant_suggest object fields:

  • text - type: str
  • srn - type: str
  • value - type: str

request example:

curl --header "Acc-Token: eyJlbXBsb3llZSI6NjEyNjEsIm1lcmNoYW50Ijo1NTEzOH0.C8di6Q.JPtyyp32oH7KaM7WqyjQPE0SoXA" -X GET "http://my.example.com:5001/remote/api/v1/merchant_suggest"

response example:

{
  "status": "ok",
  "errors": [
  ],
  "data": {
    "merchants": [
    ]
  },
  "system": ""
}
