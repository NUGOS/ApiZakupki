Просмотр закупки

GET remote/api/v1/entity/{id}/lot/{lot_id}

GET remote/api/v2/entity/{id}/lot/{lot_id}

id - id закупки из списка закупок

lot_id - id лота из lot_preview объекта

UPDATE for questions, claims, complaints in v2

  • removed questions collection from data field, moved to separate endpoint (see question_list doc)
  • removed complaints collection from data field, moved to separate endpoint (see claim_list & complaint_list docs)
  • removed claim_complaint_info from data field, moved to separate endpoint (see claim_list & complaint_list docs)
  • added questions_count, claims_count, complaints_count to lot data fields
