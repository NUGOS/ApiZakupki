Статические данные

GET remote/api/v1/static_data

GET remote/api/v2/static_data

возвращает список доступных типов закупок, областей и прочее, что использутеся для фильтров поиска

UPDATE 2018-05-11

entity_type added in v2

request headers:

  • Api-Agent - type: string, required

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

possible response data fields:

  • status - type: array of status objects, described below
  • locations - type: array of location objects, described below
  • method_type - type: array of method_type objects, described below
  • units - type: array of unit objects, described below
  • timeline_status - type: array of timeline_status objects or empty, described below
  • qualification_status - type: array of qualification_status objects, described below
  • offer_status - type: array of offer_status objects, described below
  • pre_qualification_status - type: array of pre_qualification_status objects, described below
  • lot_status - type: array of lot_status objects, described below
  • contract_status - type: array of contract_status objects, described below
  • entity_type - type: array of entity_type objects, described below only in v2
  • cashflow_types - type: array if cashflow_types objects, described below, only in v2
  • main_procurement_category - type: array of main_procurement_category objects, described below, only in v2
  • lot_participate_status - type: array of lot_participate_status objects, described below, only in v2

qualification status object fields:

  • value_label - type: str
  • value - type: int
  • label - type: str

offer status object fields:

  • value_label - type: str
  • value - type: int
  • label - type: str

prequalification status object fields:

  • value_label - type: str
  • value - type: int
  • label - type: str

lot status object fields:

  • value_label - type: str
  • value - type: int
  • label - type: str

contract status object fields:

  • value_label - type: str
  • value - type: int
  • label - type: str

status object fields:

  • value - type: int
  • label - type: str
  • is_public - type: bool, based on this value show/hide for status filter
  • timeline_status - type: str, used for time line mapping
  • value_label - type: str

location object fields:

  • id - type: int
  • value - type: str
  • label - type: str

unit object fields:

  • id - type: int
  • value - type: str
  • label - type: str

method_type object fields:

  • value - type: int
  • value_label - type: str
  • label - type: str

entity_type object fields:

  • value - type: int
  • value_label - type: str
  • label - type: str

timeline_status object fields:

  • value - type: int
  • key - type: str

cashflow_type object fields:

  • value - type: int
  • label - type: str

main_procurement_category fields:

  • value - type: str
  • label - type: str

lot_participate_status fields:

  • value - type: int
  • label - type: str

curl -X GET "http://my.example.com:5001/remote/api/v1/static_data"
