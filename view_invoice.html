Просмотр инвойса

GET remote/api/v2/view_invoice/{invoice_id}

request headers:

  • Api-Agent - type: string, required
  • Acc-Token - type: string, required

request params:

  • invoice_id - type: int, required

возвращает PDF инвойса по invoice_id (см. list invoices)

Examples

request:

curl -i -X GET "http://my.example.com:5001/remote/api/v2/view_invoice" -H "Acc-Token: 2ec2c2454b63dd5f3f96e6b6ebad7aa922f0b952"
