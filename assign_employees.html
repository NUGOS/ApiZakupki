Назначение, снятие менеджеров на закупку

Назначение/переназначение менеджера

POST remote/api/v2/managers/assign

request headers:

  • Api-Agent - type: string, required
  • Acc-Token - type: string, required

possible request arguments:

  • state_purchase_id - type: int
  • assignee_employee_id - type: int, possible values - one of users from assignment_data

possible response fields:

  • status - type: str, values: ok/error

Examples

request body:

{}

curl --header "Api-Agent: zk_android" --header "Acc-Token: eyJlbXBsb3llZSI6NjEyNjEsIm1lcmNoYW50Ijo1NTEzOH0.C8di6Q.JPtyyp32oH7KaM7WqyjQPE0SoXA" -X POST "http://my.example.com:5001/remote/api/v2/managers/assign" -d@request_body.json
