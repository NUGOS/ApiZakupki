favorites

POST remote/api/v1/entity/toggle_favorite/{id}

id - id закупки из списка закупок

request headers:

  • Api-Agent - type: string, reuqired
  • Acc-Token - type: string, required

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

possible response data fields: data response always empty object

запрос на этот endpoint меняет значение поля is_favorite между true и false

request:

curl --header "Acc-Token: eyJlbXBsb3llZSI6NjEyNjEsIm1lcmNoYW50Ijo1NTEzOH0.C-RgqA.iwxHPKJq2KHCj8Qe6BvcvJlSNss" -X GET "http://my.example.com:5001/remote/api/v1/entity/toggle_favorite/262492"

response:

{
    "status": "ok",
    "errors": [],
    "data": {},
    "system": []
}
