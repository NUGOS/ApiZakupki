Список инвойсов

GET remote/api/v2/list_invoices?page={int}

request headers:

  • Api-Agent - type: string, required
  • Acc-Token - type: string, required

постранично возвращает список инвойсов (20 на страниц) отсортированых оп дате создания (desc)

possible request arguments:

  • page - type: int, optional, 1 by default

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

possible response data fields:

  • total - type: int
  • invoices - type: array of invoice_list_item objects or empty, described below
  • current_page - type: int
  • last_page - type: int
  • is_last_page - type: bool

possible invoice_list_item object fields:

  • date_created - type: datetime string, format yyyy-mm-ddTHH:MM:SS.fZ,
  • id - type: int
  • title - type: string
  • is_fully_paid - type: bool
  • is_partial_paid - type: bool
  • is_over_paid - type: bool
  • pay_status - type: int, one of [0, 1, 2, 3, 4]

Examples

request:

curl -X GET -H "Acc-Token: token" -H "Api-Agent: zk_ios" "http://my.example.com:5001/remote/api/v2/list_invoices?page=2"
