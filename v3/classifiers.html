Classifiers (DK021 aka CPV)

GET remote/api/v3/classifiers?timestamp={timestamp || ''}

returns all available dk021(cpv) classifiers if timestamp provided - we try to find any changes since timestamp and return new classifiers data if any was, in other case empty response will be returned

request headers:

  • Api-Agent - type: string, required

possible request arguments:

  • timestamp - type: int

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

possible response data fields:

  • primary_classifier - type: array of primary_classifier objects or empty, described below
  • timestamp - type: int

primary_classifier object fields:

  • full_title - type: str
  • code - type: str
  • value - type: str
  • title - type: str

curl -X GET "http://my.example.com:5001/remote/api/v3/classifiers"
