Save user filter

POST remote/api/v3/save_filter

save search filters

request headers:

  • Api-Agent - type: string, required
  • Acc-Token - type: string, required

possible request arguments:

  • filter_name - type: string, required
  • locations - type: array of string, possible values - one or more locations.value
  • use_delivery_location - type: bool
  • primary_classifiers - type: array of str, possible values - one or more primary_classifier.value
  • amount_from - type: float
  • amount_to - type: float
  • merchant_name - type: array of string, possible values - one or more merchant_suggest.value
  • method_type - type: array of int, possible values - one or more method_type.value
  • range_date_created_dt_end - type: string, values format YYYY-MM-DD
  • range_date_tendering_start_dt_start - type: string, values format YYYY-MM-DD
  • range_date_tendering_start_dt_end - type: string, values format YYYY-MM-DD
  • range_date_tendering_end_dt_start - type: type: string, values format YYYY-MM-DD
  • range_date_tendering_end_dt_end - type: type: string, values format YYYY-MM-DD
  • range_date_auction_start_dt_start - type: type: string, values format YYYY-MM-DD
  • range_date_auction_start_dt_end - type: string, values format YYYY-MM-DD
  • status - type: array of int, possible values - one or more status.value
  • entity_type - type: array of int, possible values - one or more entity_type.value
  • assignee_employee - type: array of int, possible values - one or more idsassignment_data.available_users.id
  • re - type: int
  • in - type: int
  • fp - type: int, single, more details in search

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

possible response data fields:

  • status - type: array of status objects or empty, possible values - one or more stauses
  • method_type - type: array method_type objects or empty, possible values - one or more method_types
  • range_date_auction_start_dt_end - type: date str or null, format YYYY-MM-DD
  • range_date_tendering_end_dt_start - type: date str or null, format YYYY-MM-DD
  • locations - type: array of location objects or empty, possible values - one or more locations
  • range_date_tendering_end_dt_end - type: date str or null, format YYYY-MM-DD
  • amount_from - type: float str or null
  • amount_to - type: float str or null
  • range_date_created_dt_start - type: date str or null, format YYYY-MM-DD
  • range_date_created_dt_end - type: date str or null, format YYYY-MM-DD
  • keywords - type: array of str or empty
  • filter_id - type: int (signed)
  • range_date_auction_start_dt_start - type: date str or null, format YYYY-MM-DD
  • range_date_tendering_start_dt_end - type: date str or null, format YYYY-MM-DD
  • primary_classifier_id - type: array of primary classifier objects or empty, see classifiers
  • classifier_id - type: array, always empty
  • range_date_tendering_start_dt_start - type: date str or null, format YYYY-MM-DD
  • merchant_name - type: array of merchant_name objects or empty
  • entity_type - type: array of entity_type objects, described below
  • re - type: int
  • in - type: int
  • fp - type: int
