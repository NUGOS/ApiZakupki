Assign/unassign employee(manager) to tender

Assing

POST remote/api/v3/assign

request headers:

  • Api-Agent - type: string, required
  • Acc-Token - type: string, required

possible request arguments:

  • state_purchase_uid - type: str
  • assignee_employee_id - type: int, possible values - one of users from assignment_data

possible response fields:

  • status - type: str, values: ok/error

Example

request:

clear && curl -X POST "http://my.example.com:5001/remote/api/v3/assign" --header "Acc-Token: 43b06b921e3df14a66cb9a38be51e71dabde5856" --header "Api-Agent: zk_ios" --header "Content-Type: application/json" --data '{"state_purchase_uid": "c7d96eba9e1c49a29e5ad1f3cf4f4fd4", "assignee_employee_id": 61471}' | json_pp

response:

{
   "status" : "ok"
}

Unassign

request headers:

  • Api-Agent - type: string, required
  • Acc-Token - type: string, required

possible request arguments:

  • state_purchase_uid - type: str

possible response fields:

  • status - type: str, values: ok/error

Example

request:

clear && curl -X POST "http://my.example.com:5001/remote/api/v3/unassign" --header "Acc-Token: 43b06b921e3df14a66cb9a38be51e71dabde5856" --header "Api-Agent: zk_ios" --header "Content-Type: application/json" --data '{"state_purchase_uid": "c7d96eba9e1c49a29e5ad1f3cf4f4fd4"}' | json_pp
