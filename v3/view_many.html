View multiply tenders

GET remote/api/v3/multiply_view[?uid={purchase_uid}&uid={purchase_uid}...

purchase_uid - id of tender from view purchase or search

request headers:

  • Api-Agent - type: string, required
  • Acc-Token - type: string, required

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

possible response data fields:

  • purchases - type: array of purchases, more detail and format here
  • ignored_uids - type: array of str, uids that was ignored

Examples

request:

clear && curl -X GET "http://my.example.com:5001/remote/api/v3/multiply_view?uid=604b9ce6734bb827c49347857ba29e4a867d5136&uid=36f21b9440c341e0bffaab12161e25d1" --header "Acc-Token: 43b06b921e3df14a66cb9a38be51e71dabde5856" --header "Api-Agent: zk_ios" | json_pp
