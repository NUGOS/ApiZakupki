Get API endpoint by tender tuid or tender tuid + lot-uid

GET remote/api/v3/by_tuid/{tuid}{tail}

tuid - purchase tuid from view or search

tail - can contain any valid url string

request headers:

  • Api-Agent - type: string, reuqired
  • Acc-Token - type: string, required

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

possible data fields:

  • tender - type: str
  • lot - type: str or null

Examples:

request:

clear && curl -X GET --header "Acc-Token: 5a119647c3fa191ab60d41ed9fabf784453999d4" --header "Api-Agent: zk_ios" "http://my.example.com:5001/remote/api/v3/by_tuid/R-UA-2019-07-20-1000005-z" | json_pp

response:

{
   "data" : {
      "lot" : null,
      "tender" : "976d1f734f0c446dbfd3b78bd2d84edf"
   },
   "errors" : [],
   "status" : "ok",
   "system" : ""
}
