Lot view

GET remote/api/v3/view/{tender_uid}/lot/{lot_uid}

tender_uid - purchase uid from purchase object, required

lot_uid - lot uid from purchase lot object, required

request headers:

  • Api-Agent - type: string, required
  • Acc-Token - type: string, required

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

possible response data fields:

  • amount - type: amount object, see view purchase doc
  • assignee_supplier_employee - type: assignee supplier object or null, see view purchase doc
  • enquiry_period - type: object, state period object, see view purchase doc
  • tendering_period - type: object, state period object, see view purchase doc
  • auction_period - type: object, state period object, see view purchase doc
  • auction_url - type: str or null
  • cancellations - type: cancellation objects list, see view purchase doc
  • claims_count - type: int
  • complaints_count - type: int
  • descr - type: str or null
  • descr_en - type: str or null
  • documents - type: documents object, see view purchase doc
  • features - type: feature objects list, see view purchase doc
  • guarantee - type: guarantee object, see view purchase doc
  • id - type: str
  • is_blocked - type: bool
  • is_participate - type: bool
  • items - type: item objects list, see view purchase doc
  • lot_status - type: int
  • lot_status_text - type: str
  • offers - type: offer objects list, see view purchase doc
  • pre_qualification - type: pre_qualification objects list, see view purchase doc
  • public_link_lot - type: str
  • public_link_tender - type: str
  • qualification - type: qualification objects list or empty, see view purchase doc
  • questions_count - type: int
  • stage1_id - type: str or null
  • stage2_id - type: str or null
  • step - type: amount object, see view purchase doc
  • title - type: str
  • title_en - type: str or null
  • tender_id - type: str
  • user_offer - type: offer object or null, see view purchase doc
  • main_procurement_category - type: main_procurement_category object or null, see view purchase doc
  • milestones - type: array of milestones objects, see view purchase doc
  • cabinet_url - type: str
  • is_free - type: bool
  • max_awards_count - type: int or null
  • clarifications_until - type: str or null, format: yyyy-mm-ddTHH:MM:SS
  • complaint_period - type: object, state period object, described below
  • invalidation_date - type: str or null, format: yyyy-mm-ddTHH:MM:SS
  • agreement_duration - type: srt or null

clear && curl -X GET --header "Acc-Token: 5a119647c3fa191ab60d41ed9fabf784453999d4" --header "Api-Agent: zk_ios" "http://my.example.com:5001/remote/api/v3/view/b3c4961cc07d48cdba8b2e21462d9443/lot/227cfc866c084bc8a09e0873e236f311" | json_pp
