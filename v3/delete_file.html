Delete file

POST remote/api/v3/delete_file/{id}

delete uploaded file id - file id, can be getted from upload file response

request headers:

  • Api-Agent - type: string, required
  • Acc-Token - type: string, required

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

always returns empty success response

Example

request:

clear && curl -X POST "http://my.example.com:5001/remote/api/v3/delete_file/1246173" --header "Acc-Token: 0064e3bbac9bd39c6d40af0f16b5852b4f84da22" --header "Api-Agent: zk_ios" | json_pp

response:

{
   "data" : {},
    "errors" : [],
    "status" : "ok",
    "system" : ""
}
