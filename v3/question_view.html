View single question

GET /remote/api/v3/questions/{tuid}/question/{quid}

tuid - purchase uid from purchase view or purchase search quid - question uid from questions list

request headers:

  • Api-Agent - type: string, required
  • Acc-Token - type: string, required

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

response data fields:

  • answer - type: str or null
  • author - type: merchant object or empty, see tender view
  • date - type: date string, format YYYY-MM-DDTHH:MM:SS or null
  • date_answered - type: date string, format YYYY-MM-DDTHH:MM:SS or null
  • description - type: str
  • question_of - type: str
  • title - type: str
  • uid - type: str

Example:

request:

clear && curl -X GET "http://my.example.com:5001/remote/api/v3/questions/f06cc892fd9841fca0b4a8ae4859f4c8/question/46d0bc4ae1954233a8de6c531a46f51d" --header "Acc-Token: e53e0e4a34784eb359a9abba73b81927e2cb48fa" --header "Api-Agent: zk_ios" | json_pp
