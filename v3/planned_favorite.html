Add/remove from planned or favorites

favorite

POST remote/api/v3/favorite/{uid}

uid - uid of purchase from search or from view

request headers:

  • Api-Agent - type: string, reuqired
  • Acc-Token - type: string, required

possible request fields:

  • action - type: bool, optional. If true/false already exists do nothing, else do toggle

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

possible response data fields: data response always empty object

Example:

request:

clear && curl -X POST --header "Acc-Token: 5a119647c3fa191ab60d41ed9fabf784453999d4" --header "Api-Agent: zk_ios" "http://my.example.com:5001/remote/api/v3/favorite/bd54d665335c4d45bdb26f5e9398a5ec" --header "Content-Type: application/json" --data '{"action": true}'| json_pp

response:

{
   "data" : {},
   "errors" : [],
   "status" : "ok",
   "system" : ""
}

planned

POST remote/api/v3/planned/{uid}

uid - uid of purchase from search or from view

request headers:

  • Api-Agent - type: string, reuqired
  • Acc-Token - type: string, required

possible request fields:

  • action - type: bool, optional. If true/false already exists do nothing, else do toggle

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

possible response data fields:

  • balance_info - type: balance_info object, described below

balance_info object fields:

  • recommended_amount - type: float
  • juridical_address - type: str
  • srn - type: str
  • name - type: str
  • vat_num - type: str
  • balance - type: float

Example:

request:

clear && curl -X POST --header "Acc-Token: 5a119647c3fa191ab60d41ed9fabf784453999d4" --header "Api-Agent: zk_ios" "http://my.example.com:5001/remote/api/v3/planned/bd54d665335c4d45bdb26f5e9398a5ec" | json_pp

response:

{
   "data" : {
       "balance_info": {
            "recommended_amount": 0,
            "juridical_address": "Украина",
            "srn": "",
            "name": "Назва моєї компанії",
            "vat_num": "",
            "balance": 10000.0 
        }
   },
   "errors" : [],

}
