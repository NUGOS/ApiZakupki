API endpoint for save information to Employee about bitrix url and filter_profile_id

POST remote/api/v3/bitrix_subscription

request headers:

  • Api-Agent - type: string, required
  • Acc-Token - type: string, required

possible request arguments:

  • url - type: str
  • id - type: str

possible response fields:

  • status - type: str, values: ok/error

Examples:

request:

clear && curl -X GET --header "Acc-Token: 5a119647c3fa191ab60d41ed9fabf784453999d4" --header "Api-Agent: zk_ios" "http://my.example.com:5001/remote/api/v3/bitrix_subscription" --data '{"url": "https://bitrix/some/user/subscription/123456789?param=12345", "id": "9902"}' | json_pp

response:

{
  "data": {},
  "errors": [],
  "status": "ok",
  "system": ""
}
