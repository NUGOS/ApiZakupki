List of all user saved filters

GET remote/api/v3/saved_filters

request headers:

  • Api-Agent - type: string, required
  • Acc-Token - type: string, required

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

possible data fields:

  • saved_filters - type: array of saved_filter objects or empty, described below

saved filter fields:

  • id - type: int, signed
  • name - type: str
  • readonly - type: bool
  • selected - type: bool

Example

request:

clear && curl -X GET "http://my.example.com:5001/remote/api/v3/saved_filters" --header "Acc-Token: 43b06b921e3df14a66cb9a38be51e71dabde5856" --header "Api-Agent: zk_ios" | json_pp
