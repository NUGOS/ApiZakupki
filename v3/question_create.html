Create new question

POST /remote/api/v3/question

request headers:

  • Api-Agent - type: string, required
  • Acc-Token - type: string, required

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

possible request fields:

  • state_purchase - type: str, required
  • state_purchase_lot - type: str, optional if state_purchase passed
  • question - type: str, required
  • title - type: str, required

possible response data fields:

  • see view question

Example

request:

clear && curl -X POST "http://my.example.com:5001/remote/api/v3/question" --header "Content-Type: application/json" --header "Acc-Token: 0064e3bbac9bd39c6d40af0f16b5852b4f84da22" --header "Api-Agent: zk_ios" --data '{"state_purchase": "861092d71ce38fec32e689cb7b672c400be9256b", "question": "q", "title": "t"}' | json_pp
