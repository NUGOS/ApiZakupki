List of tender uids by search query

GET remote/api/v2/uids

request headers:

  • Api-Agent - type: string, reuqired
  • Acc-Token - type: string, required

possible request arguments: see search documentation

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

possible response data fields:

  • total_items - type: int, total found entities
  • data - type: array of tender item objects or empty, described below
  • page - type: int, current page

tender item object fields:

  • uid - type: str
  • merchant_srn - type: str

Example:

request:

clear && curl -X GET --header "Acc-Token: Token" --header "Api-Agent: zk_ios" "http://my.zakupki.prom.ua/remote/api/v2/uids?merchant_name=123123&merchant_name=5433445344" | json_pp
