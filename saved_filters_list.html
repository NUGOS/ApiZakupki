Список сохраненных фильтров пользователя

GET remote/api/v1/saved_filters

request headers:

  • Api-Agent - type: string, required
  • Acc-Token - type: string, required

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

possible data fields:

  • saved_filters - type: array of saved_filter objects or empty, described below

saved filter fields:

  • id - type: int, signed
  • name - type: str
  • readonly - type: bool
  • selected - type: bool

request example:

curl -X GET "http://my.example.com:5001/remote/api/v1/saved_filters"
