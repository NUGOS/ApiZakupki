Создание счета

POST remote/api/v2/create_invoice

request headers:

  • Api-Agent - type: string, required
  • Acc-Token - type: string, required

possible request fields:

  • amount - type: float, optional, default value = 510
  • srn - type: str, required, max length - 32
  • name - type: str, required, max length - 255
  • juridical_address - type: str, required, max length - 255
  • vat_num - type: str, optional, max length - 12

создаине инвойса

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

possible response data fields:

  • date_created - type: datetime string, format yyyy-mm-ddTHH:MM:SS.fZ,
  • id - type: int
  • title - type: string
  • is_fully_paid - type: bool
  • is_partial_paid - type: bool
  • is_over_paid - type: bool
