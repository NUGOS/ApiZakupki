Получить ссылку на тендер по его tuid (UA-XXXX-XX-XX-XXXXX-x)

GET remote/api/v1/view_by_tuid/{tuid}{tail}

tuid - tuid закупки из списка закупок

tail - может содержать любую валидную url строку

request headers:

  • Api-Agent - type: string, reuqired
  • Acc-Token - type: string, required

possible response fields:

  • status - type: str, values: ok/error
  • errors - type: array of objects, validation errors
  • data - type: object
  • system - type: str, system error messages

possible data fields:

  • url - type: str

request:

curl --header "Acc-Token: eyJlbXBsb3llZSI6NjEyNjEsIm1lcmNoYW50Ijo1NTEzOH0.C8di6Q.JPtyyp32oH7KaM7WqyjQPE0SoXA" -X GET "http://my.example.com:5001/remote/api/v1/view_by_tuid/UA-8340-63-45-811473-1"

success response (no lots):

{
  "status": "ok",
  "errors": [
  ],
  "data": {
    "url": "remote/api/v1/entity/262485"
  },
  "system": ""
}
